<h2> 🏡 프로젝트 개요 </h2>
  <p> 식집사는 환경 보호 활동을 통해 식물을 키우는 타이쿤 게임입니다.</p>
  <p> 환경 보호 미션을 클리어 할 때마다 식물이 쑥쑥 자라는 걸 볼 수 있어요. </p>
