/ 파일 open 방법/
  - 제일 위 * 압축 해제 후 안드로이드 스튜디오 프로그램에서 open시 PlantButler -> PlantButler 파일을 열어서 실행해야 합니다

1. 로그인
 1) 회원가입 시 입력했던 이메일과 비밀번호로 로그인한다.

 2) 비밀번호를 분실했을 때에는 하단의 '비밀번호를 잃어버리셨나요?'를 선택한다.
   (1) 사용자의 이메일을 입력하면, 사용자의 비밀번호가 출력된다. 

 3) 회원가입을 원할 시에는 하단의 '회원가입' 버튼을 선택한다.

2. 회원가입
 1) 회원가입 시 이메일, 비밀번호, 닉네임을 입력한다.

 2) 비밀번호는 2번 입력하게 되는데, 이때 첫번째 입력한 비밀번호와 재차 입력한 비밀번호가 일치하는지 '확인' 버튼을 통해 확인할 수 있다.

 3) 비밀번호 입력 시에는 6자가 넘어야 회원가입이 가능하다.   

3. 메인 화면
 1) 첫 로그인 시에 메인화면에 입장하면 '+' 버튼이 존재한다.
   (1) 해당 버튼을 누르면, 사용자가 기를 식물을 선택할 수 있다.
   (2) 두 식물 중, 하나를 선택한 후 식물의 이름을 입력한다.

 2) 우측 상단에는 사용자의 걸음 수를 확인할 수 있다.
   (1) 사용자의 걸음 수 위에는 설정 버튼이 존재한다.

 3) 좌측 상단에는 '오늘의 퀴즈 풀기' 버튼이 존재한다.
   (1) 오늘의 퀴즈는 사용자당 하루에 한 번으로 제한된다.

 4) '오늘의 퀴즈 풀기' 버튼 위에는 사용자의 포인트를 확인할 수 있다.

 5) 하단에는 마이페이지, 커뮤니티, 상점, 미션 버튼이 존재한다.

 3) 식물 성장도
   (1) 식물은 아이템으로 성장할 수 있고, 100%까지 성장할 수 있다.

4. 설정 버튼
 1) 프로필 수정 버튼
   (1) 현재 사용자의 프로필사진, 닉네임, 이메일, 비밀번호가 출력된다.
        1 해당 정보는 수정 가능하다. 수정하고 싶은 정보 옆의 연필 모양 버튼을 선택한다.
        2 변경할 정보를 두 번 입력한 후, 설정완료 버튼을 누르면 정보가 변경된다.

 2) 로그아웃 버튼
   (1) 로그아웃 버튼을 누를 시, 현재 로그인되어 있던 계정에서 로그아웃 되고 로그인 화면으로 돌아간다.

 3) 탈퇴 버튼
   (1) 탈퇴 버튼을 누를 시, 현재 로그인되어 있던 계정이 탈퇴되고 해당 계정으로 로그인이 불가하다.

5. 마이페이지
: 사용자가 키운 식물이 모두 출력된다. 하단의 키운 식물 모음에는 현재 키우고 있는 식물도 포함된다.
1)현재 키우고 있는 식물 버튼
   (1)현재 키우고 있는 식물을 누를 시, 현재 구매한 흙, 영양제 아이템들을 사용할 수 있다.
   (2)아이템을 사용하면 아이템 효과에 맞게 현재 키우고 있는 식물의 경험치가 올라간다.
2)하단의 키운 식물 버튼
   (1)하단의 키운 식물을 누를 시, 현재 구매한 화분 아이템을 적용할 수 있다.
   (2)꾸미고자 하는 화분을 선택하면 선택한 식물의 화분이 바뀌어 보인다.
   (3)저장버튼을 누를 시, 해당 식물의 화분이 변경된다.

6. 커뮤니티
 1) 나만의 정원
   (1) 하단에 사용자가 키운 식물이 모두 출력된다.
   (2) 식물을 선택하면 상단의 5개의 칸에 출력되고, '저장'버튼을 누르면 '나만의 정원'이 저장된다.
   (3) 액자에는 5개의 식물이 모두 차있어야 하므로, 1개의 식물을 기르고 있더라도 5번 클릭하여 모든 식물을 '나만의 정원'에 저장한다.

 2) 게시판
   (1) 게시판에는 식집사 어플을 사용하는 사용자들이 게시한 글이 출력된다.
       1 게시판에서 특정 게시글을 선택하면 해당 게시글을 작성한 사용자의 닉네임과, 게시글 제목, 내용, 하트 수가 출력된다.
       2 게시글 목록 오른쪽 하트나 게시물 아래의 하트를 클릭하면 사용자 게시글의 하트 수가 증가한다.

   (2) 사용자가 게시글을 업로드하기 위해서는 우측 하단의 '글쓰기' 버튼을 클릭한다.
        1 제목과 글 내용을 입력한 후 '등록' 버튼을 클릭하여 저장한다.

 3) 이달의 식집사
    (1) 식집사 어플을 사용하는 사용자의 랭킹이 출력된다.
    (2) 해당 랭킹은 게시물의 하트수로 정렬되어 순위를 측정한다.
    (3) 랭킹에 있는 프로필을 선택하면 해당 사용자의 '나만의 정원'을 확인할 수 있다.

 4) 뉴스기사
    (1) 환경과 관련된 뉴스가 출력된다.
    (2) 특정 뉴스를 선택하면, 인터넷의 원문 기사로 넘어간다.

7. 상점
 : 흙, 영양제, 화분을 사용자가 가진 포인트로 구매하 수 있다.
사용자가 가진 포인트가 부족할 경우 포인트가 부족하다는 토스트 메시지를 출력한다.
1)흙 상점  
    (1) 사용자가 보유한 포인트로 흙 아이템을 중복 구매할 수 있다. 
    (2) 구매한 아이템의 효과에 따라 사용자가 현재 키우고 있는 식물의 성장도를 증가시킬 수 있다.
    (3) 구매한 아이템은 마이페이지에서 사용 가능하다.
2)영양제 상점
    (1) 사용자가 보유한 포인트로 영양제 아이템을 중복 구매할 수 있다. 
    (2) 구매한 아이템의 효과에 따라 사용자가 현재 키우고 있는 식물의 성장도를 증가시킬 수 있다.
    (3) 구매한 아이템은 마이페이지에서 사용 가능하다.
3)화분 상점
    (1) 사용자가 보유한 포인트로 화분 아이템을 구매할 수 있다. 단, 중복 구매는 불가능 하다.
    (2) 구매한 화분 아이템은 사용자가 키운 식물에 적용할 수 있다.
    (3) 구매한 아이템은 마이페이지에서 사용 가능하다.


8. 미션
 : 미션 목록에는 일일 미션, 주간 미션, 선택 미션, 미션 검사가 존재한다.
 1) 일일 미션
    (1) 사용자의 걸음 수를 통해 미션을 확인받을 수 있다. 만약 사용자의 걸음 수가 충족하지 않고 '미션확인' 버튼을 클릭하면 '미션실패'가 토스트로
	출력된다.
    (2) 사용자의 걸음 수를 충족하고 '미션확인' 버튼을 클릭하면 해당 미션의 포인트만큼 획득할 수 있다.

 2) 주간 미션
    (1) 일일 미션과 동일하다.

 3) 선택 미션
    : 선택 미션에는 대중교통 이용, 쓰레기 줍기, 텀블러 사용, 장바구니 사용, 분리수거, 기타가 존재한다
    (1) 미션을 선택하면, 해당 선택 미션을 수행한 것을 등록하기 위한 액티비티가 출력된다.
    (2) 갤러리 버튼을 클릭하여, 휴대폰 내의 갤러리에 존재하던 사진을 업로드할 수 있다.
    (3) 사진을 업로드 한 후, 등록 버튼을 클릭하면 시스템에 저장된다.
    (4) 해당 사진은 미션 검사 목록에서 다른 사용자들이 검토한 후, 인증이 되면 경험치를 획득할 수 있다.

 4) 미션 검사
    (1) 식집사 어플을 사용하는 사용자들이 등록한 사진 중 랜덤으로 9개가 출력된다. 그 중, 환경친화적인 사진이라고 생각되는 사진을 선택한다.
    (2) '신고하기' 버튼을 선택하면, 미션 인증 사진 중 신고할 사진을 선택할 수 있다.
    (3) 다른 사용자가 신고한 사진을 등록한 사용자는 경험치와 포인트가 0이 된다.
